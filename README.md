# Welcome to SPCEDES - FRONTEND!

### Stack tech

1. **React**: Libreria UI moderno para crear una SPA interactiva.
    - Fetch: Axios
    - Forms: React Hook forms
    - State manager: Redux
    - Testing: Vitest + Testing Librery 
    - Auth: Auth.js -> Hay que esperar para instalar esta en OPEN PR
    - Animations: Framer Motion
2. **Remix**: Mas fácil para el despliegue en la nube, plan B vercel 
4. **ShadCN**: Excelente para crear interfaces estilizadas con un diseño moderno y componentes reutilizables.
5. **TailwindCSS**: Sistema de utilidades CSS que permite estilos rápidos y consistentes.

• **Vercel**: Plataforma optimizada para aplicaciones Next.js con despliegues automáticos y CDN integrado. Ideal para hospedar el frontend.

Run script:

```bash
$ bun dev

$ bun start

$ bun lint

$ bun format
```
